# Package index

## Installation and update

- [`easystats_packages()`](https://easystats.github.io/easystats/reference/easystats_packages.md)
  : List all packages in the easystats ecosystem

- [`install_latest()`](https://easystats.github.io/easystats/reference/install_latest.md)
  : Install the easystats suite from R-universe (GitHub) or CRAN

- [`install_suggested()`](https://easystats.github.io/easystats/reference/install_suggested.md)
  [`show_suggested()`](https://easystats.github.io/easystats/reference/install_suggested.md)
  [`show_reverse_dependencies()`](https://easystats.github.io/easystats/reference/install_suggested.md)
  : Download all suggested packages

- [`easystats_update()`](https://easystats.github.io/easystats/reference/easystats_update.md)
  :

  Update *easystats*-packages and its dependencies from CRAN, if
  necessary.
