# List all packages in the easystats ecosystem

List all packages in the easystats ecosystem

## Usage

``` r
easystats_packages()
```

## Value

A character vector

## Examples

``` r
easystats_packages()
#>  [1] "bayestestR"  "correlation" "datawizard"  "easystats"   "effectsize" 
#>  [6] "insight"     "modelbased"  "performance" "parameters"  "report"     
#> [11] "see"        
```
